# 10 заданий для начинающих верстальщиков по HTML/CSS

Задания сфокусированы на работе с отступами: `padding` и `margin`. В каждом задании есть конкретная структура, свойства для тренировки и вопрос для самопроверки.

---

## Задание 1. Страница «Обо мне» с внешними отступами

**Цель:** научиться управлять расстоянием между элементами через `margin`.

Создай HTML-страницу. Внутри `<body>` размести:
- `<h1>` — своё имя;
- `<h2>` — название профессии или хобби;
- `<h3>` — любимая цитата;
- `<p>` — 2–3 предложения о себе.

**Что нужно сделать:**
- каждому заголовку задай свой `font-size`;
- заголовкам `<h1>`, `<h2>`, `<h3>` задай разный `margin-bottom`, чтобы расстояние между заголовком и следующим элементом было разным;
- абзацу `<p>` задай `margin-top` и `margin-bottom`, чтобы он отодвинулся и от заголовка, и от нижнего края страницы;
- странице задай `background` светлого цвета, а тексту — `color` тёмного оттенка.

**Теги:** `h1`, `h2`, `h3`, `p`  
**Свойства CSS:** `font-size`, `color`, `background`, `margin`

**Вопрос для самопроверки:** что изменится, если у заголовка убрать `margin-bottom`, а у следующего элемента убрать `margin-top`?

## Задание 2. Кнопка-ссылка с внутренними отступами

**Цель:** понять разницу между `padding` (внутренние отступы) и `margin` (внешние отступы).

Создай ссылку `<a>` с текстом «Заказать звонок». Она должна выглядеть как кнопка.

**Что нужно сделать:**
- ссылке задай `background`, `color` и `border-radius`, чтобы она стала похожа на кнопку;
- через `padding` сделай так, чтобы текст не прилипал к краям кнопки (попробуй `padding: 12px 24px`);
- через `margin` отодвини кнопку от краёв экрана и от соседних элементов;
- добавь вторую ссылку «Подробнее» и задай между кнопками `margin-right` у первой или `margin-left` у второй;
- сделай так, чтобы при наведении цвет кнопки менялся.

**Теги:** `a`  
**Свойства CSS:** `padding`, `margin`, `background`, `color`, `border-radius`

**Вопрос для самопроверки:** что произойдёт, если вместо `padding` увеличить `font-size`? Почему кнопка всё равно будет выглядеть иначе?

## Задание 3. Карточка товара с многослойными отступами

**Цель:** научиться задавать отступы одновременно родителю и дочерним элементам.

Сверстай карточку товара. Структура:

```html
<div class="card">
  <img src="..." alt="Товар">
  <h3>Название товара</h3>
  <p>Краткое описание товара в несколько строк.</p>
  <a href="#">Купить</a>
</div>
```

**Что нужно сделать:**
- `.card` задай `width`, `border`, `border-radius`, `background` и `padding` внутри карточки;
- картинке задай `width: 100%`, `border-radius` и `margin-bottom`, чтобы отодвинуть заголовок от картинки;
- заголовку `<h3>` задай `margin-bottom`, чтобы отодвинуть описание;
- описанию `<p>` задай `margin-bottom`, чтобы отодвинуть кнопку;
- кнопке `<a>` задай `padding` и `background`, сделай её блочной или строчно-блочной.

**Теги:** `div`, `img`, `h3`, `p`, `a`  
**Свойства CSS:** `width`, `padding`, `margin`, `border`, `border-radius`, `background`, `color`

**Вопрос для самопроверки:** почему важно, чтобы внутренний отступ был у `.card`, а не у каждого элемента в отдельности? Как будет выглядеть карточка без `padding` у `.card`?

## Задание 4. Список покупок с отступами между пунктами

**Цель:** разобраться с отступами внутри списка.

Создай маркированный список покупок из 5 пунктов:

```html
<ul>
  <li>Хлеб</li>
  <li>Молоко</li>
  <li>Яблоки</li>
  <li>Сыр</li>
  <li>Чай</li>
</ul>
```

**Что нужно сделать:**
- списку `<ul>` задай `background`, `padding` со всех сторон и `border-radius`;
- каждому `<li>` задай `margin-bottom`, чтобы пункты не слипались;
- последнему пункту убери нижний отступ через дополнительный класс;
- тексту внутри `<li>` задай `color` и `font-size`;
- попробуй задать `padding-left` у `<li>` и посмотри, как изменится положение текста относительно маркера.

**Теги:** `ul`, `li`  
**Свойства CSS:** `padding`, `margin`, `background`, `border-radius`, `color`, `font-size`

**Вопрос для самопроверки:** что изменится, если `margin-bottom` задать не `<li>`, а `<ul>`? Почему это работает по-другому?

## Задание 5. Топ-5 фильмов с разными отступами

**Цель:** использовать разные виды отступов в одном списке.

Создай нумерованный список топ-5 фильмов:

```html
<ol>
  <li><strong>Название фильма</strong> — краткое описание.</li>
  ...
</ol>
```

**Что нужно сделать:**
- списку `<ol>` задай `padding-left`, чтобы цифры не прилипали к краю экрана;
- каждому `<li>` задай `margin-bottom`, чтобы пункты списка дышали;
- элементам `<strong>` с названием фильма задай `margin-right`, чтобы отодвинуть текст описания;
- списку задай `background` и `padding` внутри;
- описанию внутри `<li>` задай `font-size` меньше, чем у названия.

**Теги:** `ol`, `li`, `strong`  
**Свойства CSS:** `padding`, `margin`, `background`, `color`, `font-size`

**Вопрос для самопроверки:** можно ли вместо `margin-right` у `<strong>` использовать пробелы в HTML? В чём разница?

## Задание 6. Навигационное меню с отступами между ссылками

**Цель:** научиться делать меню, где удобно управлять расстоянием между пунктами.

Создай меню из 4 ссылок:

```html
<div class="menu">
  <a href="#">Главная</a>
  <a href="#">Услуги</a>
  <a href="#">Портфолио</a>
  <a href="#">Контакты</a>
</div>
```

**Что нужно сделать:**
- `.menu` задай `background`, `padding` внутри и `border-radius`;
- ссылкам задай `color`, убери подчёркивание, задай `font-size`;
- ссылкам задай `padding` со всех сторон, чтобы увеличить кликабельную область;
- между ссылками задай отступ через `margin-right`;
- последней ссылке убери правый `margin` через класс;
- добавь эффект при наведении: изменение `background` или `color`.

**Теги:** `div`, `a`  
**Свойства CSS:** `padding`, `margin`, `background`, `border-radius`, `color`, `font-size`

**Вопрос для самопроверки:** почему для пунктов меню лучше использовать `margin-right`, чем добавлять пробелы или переносы строк между ссылками в HTML?

## Задание 7. Статья из секций с отступами между блоками

**Цель:** научиться группировать контент в блоки и задавать отступы между ними.

Сверстай статью из трёх секций:

```html
<div class="section">
  <h2>Заголовок 1</h2>
  <p>Текст первого раздела.</p>
  <ul>
    <li>Пункт 1</li>
    <li>Пункт 2</li>
  </ul>
</div>
<div class="section">...</div>
<div class="section">...</div>
```

**Что нужно сделать:**
- каждой `.section` задай `background`, `border`, `border-radius`, `padding` внутри и `margin-bottom` снаружи;
- заголовкам `<h2>` задай `margin-bottom`, чтобы отодвинуть текст;
- абзацам `<p>` задай `margin-bottom`, чтобы отодвинуть список;
- пунктам `<li>` задай `margin-bottom` и `padding-left`;
- последнему элементу в каждой секции (списку или абзацу) убери нижний отступ.

**Теги:** `div`, `h2`, `p`, `ul`, `li`  
**Свойства CSS:** `padding`, `margin`, `border`, `border-radius`, `background`

**Вопрос для самопроверки:** что лучше — задавать `margin-bottom` каждой `.section` или `margin-top` каждой следующей секции? Почему?

## Задание 8. Галерея карточек с отступами

**Цель:** потренировать отступы в повторяющихся блоках.

Сделай галерею из 4 карточек. Структура каждой карточки:

```html
<div class="gallery">
  <div class="card">
    <img src="..." alt="...">
    <p>Подпись к картинке</p>
  </div>
  ...
</div>
```

**Что нужно сделать:**
- `.gallery` задай `background` и `padding`, чтобы карточки не прилипали к краю;
- `.card` задай `width`, `border`, `border-radius`, `padding` внутри и `margin-bottom` / `margin-right`, чтобы карточки не слипались;
- картинкам задай `width: 100%`, `height`, `border-radius` и `margin-bottom`;
- подписям `<p>` задай `margin-top`, `margin-bottom` и `text-align: center`;
- попробуй расположить карточки в ряд (подсказка: `display: inline-block` для `.card`).

**Теги:** `div`, `img`, `p`  
**Свойства CSS:** `width`, `height`, `padding`, `margin`, `border`, `border-radius`, `background`, `font-size`

**Вопрос для самопроверки:** почему при использовании `inline-block` между карточками может появляться лишнее пространство? Как от него избавиться?

## Задание 9. Цитата с форматированием и отступами

**Цель:** использовать отступы внутри блока с текстом разной степени важности.

Создай блок цитаты:

```html
<div class="quote">
  <p><strong>Важная мысль:</strong> <em>жизнь — это то, что происходит, пока вы строите планы.</em></p>
  <p>Иногда стоит просто остановиться и оглянуться вокруг.</p>
  <p><s>Старый совет:</s> <u>Новый совет:</u> делай маленькими шагами.</p>
</div>
```

**Что нужно сделать:**
- `.quote` задай `background`, `border-left`, `border-radius`, `padding` со всех сторон и `margin` снаружи;
- каждому `<p>` внутри цитаты задай `margin-bottom`, чтобы абзацы не слипались;
- последнему `<p>` убери нижний отступ;
- `<strong>` задай `color` и `margin-right`;
- `<em>` задай `font-size` и `margin` по бокам;
- `<s>` и `<u>` сделай разных цветов.

**Теги:** `div`, `p`, `strong`, `em`, `s`, `u`  
**Свойства CSS:** `padding`, `margin`, `border`, `border-radius`, `background`, `color`, `font-size`

**Вопрос для самопроверки:** почему у `.quote` лучше задать `padding`, чем задавать `margin` каждому `<p>` внутри?

## Задание 10. Личная визитка с полным контролем отступов

**Цель:** закрепить работу со всеми видами отступов в одном проекте.

Сверстай личную визитку по центру экрана:

```html
<div class="card">
  <img src="..." alt="Фото">
  <h2>Имя Фамилия</h2>
  <h3>Профессия</h3>
  <p>Краткое описание о себе в 2–3 предложения.</p>
  <a href="#">Telegram</a>
  <a href="#">GitHub</a>
</div>
```

**Что нужно сделать:**
- `.card` задай `width`, `background`, `border`, `border-radius` и `padding` внутри карточки;
- картинке задай `width`, `height`, `border-radius: 50%` и `margin-bottom`, чтобы отодвинуть имя;
- `<h2>` задай `margin-bottom`, чтобы отодвинуть профессию;
- `<h3>` задай `margin-bottom`, чтобы отодвинуть описание;
- `<p>` задай `margin-bottom`, чтобы отодвинуть ссылки;
- ссылкам задай `padding` внутри, `margin-right` между ними и `border-radius`;
- странице задай `padding`, чтобы визитка не прилипала к краям экрана;
- сделай так, чтобы визитка была визуально по центру (можно через `margin: 0 auto`).

**Теги:** `div`, `img`, `h2`, `h3`, `p`, `a`  
**Свойства CSS:** `width`, `height`, `padding`, `margin`, `border`, `border-radius`, `background`, `color`, `font-size`

**Вопрос для самопроверки:** как сделать так, чтобы внутренние отступы карточки не увеличивали её общую ширину? Подсказка: свойство `box-sizing`.
